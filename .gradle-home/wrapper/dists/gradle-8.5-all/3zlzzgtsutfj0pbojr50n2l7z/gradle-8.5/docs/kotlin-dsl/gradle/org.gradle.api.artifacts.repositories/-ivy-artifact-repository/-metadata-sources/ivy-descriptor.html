ivyDescriptor

abstract fun ivyDescriptor()(source)

Indicates that this repository will contain Ivy descriptors. If the Ivy file contains a marker telling that Gradle metadata exists for this component, Gradle will also look for the Gradle metadata file. Gradle module metadata redirection will not happen if ignoreGradleMetadataRedirection() has been used.
